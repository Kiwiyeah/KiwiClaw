# KiwiClaw Admin 交付包

本仓库用于落实 OpenClaw 官方最佳实践，并输出可交付的管理员级别部署与运维文档。

## 目标
- 以本机部署为基线，形成可复用的管理员交付模板
- 覆盖安全基线、运维标准化与审计留存

## 交付文档
- `DEPLOYMENT.md`：部署与配置基线说明
- `STATUS.md`：现状盘点快照（脱敏）
- `SOP.md`：运维SOP与故障处理
- `AUDIT.md`：审计与留存策略
